# Open Graph Image

**Recommended:** Add an Open Graph image (`og-image.jpg`) to this directory for better social media sharing.

## Specifications

- **Filename:** `og-image.jpg`
- **Dimensions:** 1200x630 pixels (recommended)
- **Format:** JPG or PNG
- **Size:** Under 8MB

## Usage

This image will be displayed when your site is shared on:
- Facebook
- LinkedIn
- Twitter/X
- Other social media platforms

## Design Tips

1. Include your name and title
2. Use the green color theme (#009688) to match your site
3. Keep text readable at small sizes
4. Use a simple, clean design

## Tools to Create OG Images

- **Canva:** https://www.canva.com/templates/s/og-image/
- **Figma:** Use 1200x630px canvas
- **Adobe Express:** https://www.adobe.com/express/feature/create/og-image
- **Online generators:** Search for "open graph image generator"

## Note

For now, the meta tags reference `/assets/og-image.jpg` but this file doesn't exist yet.
Social platforms will fall back to your favicon or generate a preview from your site content.
